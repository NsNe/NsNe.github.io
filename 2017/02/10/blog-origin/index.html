博客由来

发表于 2017-02-10 | 分类于 随笔 | | 阅读次数

  不知不觉间2016已经过去，迎来了2017的日子，年少无知的我还以为自己年纪轻轻的时候，却又老了一岁，在奔三的路上又迈进了一步。古人云：吾日三省吾身，对于我这个懒人，一年省一次吧。

  回首16年，吉他没学会，React一知半解，Avalon，Regular也懵逼，不禁问自己还能干啥?只会煮煮泡面，炒炒西红柿鸡蛋？近两年的工作也是值得肯定的，也不是白干的，起码前端我会Angular，html，css呀，后端会Python，Java呀，还会shell呢，2017好好钻研React吧，钻研好了，迎娶白富美，当上CEO，走上人生巅峰指日可待，在此也感谢各位大爷大妈哥哥姐姐弟弟妹妹对小弟的关照和帮助，等我走上人生巅峰，我是不会忘了（做鬼也不会放了）你们的。

  在web开发道路上走了这么久，看过不少别人的博客，有的是在校的学生，顿时觉得自己太out了，连个博客都没有，于是想着建立一个博客，就用Hexo + github page折腾了一个。在博客上记录自己的生活学习，希望和大家共同交流web技术，共同进步，

  最后附自己大学，以及工作之后公司的照片，忆往昔峥嵘岁月稠。
